Internet Basics

What is the Internet?

The internet is a global system of interconnected computer networks which links devices worldwide. It is, in essence, a network of networks. It is linked by a multitude of optical, wireless and electronic networking technologies.

What is a Browser?

Internet Browser is a software application used to access the internet and the world wide web. Every webpage on the internet is accessible. It is done by establishing a connection between the web server and your device to display it on the same.
